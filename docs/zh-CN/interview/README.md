# Java 面试与复习

本区面向准备校招、社招、内部晋升或阶段复盘的学习者，帮助把课程知识转化为可表达、可推理、可落地的答案。

## 收录标准

- 题目必须对应已发布课程、手册条目或真实工程能力。
- 答案应包含核心结论、推理过程、常见追问和易错点。
- 涉及代码的题目以 Java 21 为验证基线。
- 只有 `learning-ready` 或 `verified` 内容可以进入默认导航。

空的 `planned` 主题不要放进默认导航；面试区不收录只有标题、没有答案或没有追问价值的占位内容。
