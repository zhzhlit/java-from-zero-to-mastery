# v0.3.8 Date Time API Design

## Goal

Continue the Java Core stage with one learning-ready date and time API lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `StudySchedule` to `examples/java-core`.
- Demonstrate `LocalDate`, `LocalTime`, `LocalDateTime`, `Duration`, `ChronoUnit`, `DateTimeFormatter`, fixed-date tests, and read-only returns.
- Extend `exercises/object-oriented` with a starter skeleton, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/09-date-time-api.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.8-SNAPSHOT`.

## Out of Scope

- Time zones, `Instant`, `ZonedDateTime`, clock injection, localization, calendar systems, and business holiday rules.
- Streams, concurrency, database persistence, schedulers, and cron expressions.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
