# v0.3.10 Enums Standard Library Design

## Goal

Continue the Java Core stage with one learning-ready enums and common standard library lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `StudyTaskStatus`, `StudyTaskPriority`, `StudyTask`, and `StudyTaskBoard` to `examples/java-core`.
- Demonstrate `enum` constants, enum fields and methods, `Optional`, `Objects`, `StringJoiner`, `Comparator`, `EnumMap`, read-only returns, and focused null handling.
- Extend `exercises/object-oriented` with starter skeletons, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/11-enums-standard-library.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.10-SNAPSHOT`.

## Out of Scope

- Records, sealed classes, annotations, reflection, localization frameworks, advanced `Optional`, and collection performance tuning.
- Java Core capstone, concurrency, database persistence, and web scheduling.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
