# v0.3.6 Generics Design

## Goal

Continue the Java Core stage with one learning-ready generics lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `LearningQueue<T>` and `ReviewResultSelector` to `examples/java-core`.
- Demonstrate generic classes, generic methods, bounded type parameters, type-safe return values, and read-only collection returns.
- Extend `exercises/object-oriented` with starter skeletons, reference solutions, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/07-generics.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.6-SNAPSHOT`.

## Out of Scope

- Wildcards, PECS, covariance and contravariance in depth, type erasure internals, reflection with generics, and advanced generic API design.
- I/O, time API, streams, records, sealed classes, and pattern matching.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
