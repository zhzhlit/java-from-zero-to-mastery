# v0.4.3 算法入门设计

## 目标

新增一个 `exercises/algorithms-basics` 练习模块和对应中文课程章节，让学生在数据结构基础之后，用测试驱动方式理解搜索、排序和 Big O 的第一层直觉。

本版本不做刷题平台，也不覆盖递归、回溯、图论或动态规划。重点是让学习者能读懂常见算法的输入、输出、边界条件和复杂度差异。

## 方案

沿用现有练习体系的 `starter` / `solution` 双模块结构。`starter` 提供可编译骨架和带 `@Disabled` 的测试练习，`solution` 提供完整参考实现和完整测试。

课程章节会把算法放回 Java 工程上下文：什么时候线性扫描足够，什么时候必须先保证数据有序，排序为什么会改变后续查找成本，以及 Big O 如何帮助我们描述输入规模增长后的行为。

## 练习范围

- `LinearSearch`：顺序查找，覆盖找到、找不到、重复值返回第一个索引和空数组。
- `BinarySearch`：二分查找，强调输入必须有序，覆盖左右边界、找不到和空数组。
- `SelectionSort`：选择排序，练习每轮选择最小值、交换和保持结果有序。
- `InsertionSort`：插入排序，练习局部有序、元素右移和小数组场景。
- `ComplexityExplorer`：用简单计数方法返回固定次数、线性次数和平方次数，帮助学习者感受 `O(1)`、`O(n)` 和 `O(n^2)`。
