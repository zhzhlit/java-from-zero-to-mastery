# v0.4.2 数据结构基础设计

## 目标

新增一个 `exercises/data-structures-basics` 练习模块和对应中文课程章节，让学生在 JUnit 5 专项练习之后，用测试驱动方式理解数组、链表、栈、队列和 Map 的基础行为。

## 方案

沿用现有练习体系的 `starter` / `solution` 双模块结构。`starter` 提供可编译骨架和带 `@Disabled` 的测试练习，`solution` 提供完整参考实现和完整测试。课程章节不追求大学教材式完整覆盖，重点放在 Java 工程实践中最常见的结构选择和边界测试。

## 练习范围

- `DynamicIntArray`：动态数组、扩容、按索引访问、删除左移、越界异常。
- `LinkedStudyList`：单向链表、头尾添加、删除头节点、遍历为只读列表。
- `StudyStack`：后进先出、查看栈顶、空栈边界。
- `StudyQueue`：先进先出、查看队首、空队列边界。
- `SimpleStringIntMap`：key/value 语义、覆盖旧值、删除、key 集合。
