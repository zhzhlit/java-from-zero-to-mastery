# v0.4.6 数据库基础设计

## 目标

新增 `exercises/database-basics` 练习模块和中文课程章节，让学生在操作系统基础之后进入数据库与 Web 阶段，先掌握表结构、SQL 标识符、参数化查询、分页和结果映射的第一层工程直觉。

本版本不连接真实数据库、不引入 JDBC 驱动，也不做事务和索引优化。练习只覆盖可稳定测试的 SQL 规则和 Java 边界代码。

## 范围

- `SqlIdentifier`：校验表名和列名，拒绝空白、非法字符和常见保留字。
- `TableSchema`：用列定义生成 `CREATE TABLE`，识别必填列和重复列。
- `SelectQueryBuilder`：构造带条件、排序、分页和参数列表的 `SELECT`。
- `InsertStatementBuilder`：构造带占位符和有序参数的 `INSERT`。
- `CourseRowMapper`：把数据库行 `Map<String, Object>` 映射成课程摘要对象。
