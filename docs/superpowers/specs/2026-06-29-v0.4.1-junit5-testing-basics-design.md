# v0.4.1 JUnit 5 专项练习设计

## 目标

新增一个可独立验证的 `exercises/testing-basics` 练习模块，让学生在刚学完测试、Maven 和工程化验证后，亲手练习 JUnit 5 的基础断言、异常测试、参数化测试、测试命名和边界条件测试。

## 方案

采用现有练习体系的 `starter` / `solution` 双模块结构。`starter` 提供可编译的待实现业务代码和带 `@Disabled` 的测试练习骨架，避免全仓验证因为学习者尚未完成练习而失败。`solution` 提供完整参考实现和完整 JUnit 5 测试。

## 练习范围

练习使用三个小对象控制复杂度：

- `StudySession`：主题、计划分钟数、摘要、专注学习块判断和构造器校验。
- `ScoreGrade`：按 0 到 100 分数边界返回 `EXCELLENT`、`PASS` 或 `RETRY`。
- `LearningPlan`：汇总学习分钟数、选择最长学习块、大小写无关主题查询。

## 文档范围

新增中文课程章节 `03-engineering-foundation/02-junit5-testing-basics.md`，并更新网站导航、路线图、仓库 README 和练习 README。文档需要明确练习命令和推荐顺序。

## 验证

交付前运行：

- `mvn -B clean verify`
- `npm run check:links --prefix website`
- `npm run docs:build --prefix website`
