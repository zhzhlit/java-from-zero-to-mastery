# v0.3.2 Inheritance and Polymorphism Design

## Goal

Continue the Java Core stage with one learning-ready inheritance and polymorphism lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `LearningResource`, `VideoResource`, and `ArticleResource` to `examples/java-core`.
- Demonstrate abstract classes, `extends`, `super`, `@Override`, final template methods, and parent-reference polymorphism.
- Extend `exercises/object-oriented` with starter skeletons, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/03-inheritance-and-polymorphism.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.2-SNAPSHOT`.

## Out of Scope

- Interfaces as a standalone topic.
- Exceptions as a standalone topic.
- Collections, generics, records, sealed classes, and pattern matching.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
