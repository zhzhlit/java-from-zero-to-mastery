# v0.4.7 JDBC 基础设计

## 目标

新增 `exercises/jdbc-basics` 练习模块和中文课程章节，让学生在数据库基础之后使用真实 JDBC API 连接 H2 内存数据库，掌握连接、参数化语句、结果映射、资源关闭和异常转换。

## 范围

- `DatabaseConfig`：保存 JDBC 连接配置。
- `ConnectionFactory`：使用 `DriverManager` 打开连接。
- `CourseRowMapper`：从 `ResultSet` 映射 `CourseRecord`。
- `JdbcCourseRepository`：实现按 id 查询、筛选已发布课程和插入课程。
- `JdbcAccessException`：转换底层 JDBC 异常。

## 验证

- `mvn -B clean verify`
- `npm run check:links --prefix website`
- `npm run docs:build --prefix website`
