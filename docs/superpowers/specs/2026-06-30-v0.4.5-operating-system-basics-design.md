# v0.4.5 操作系统基础设计

## 目标

新增 `exercises/operating-system-basics` 练习模块和中文课程章节，让学生在网络基础之后建立进程、CPU 时间片、内存单位、文件权限和环境变量的第一层工程直觉。

本版本不读取真实系统进程、不修改文件权限，也不依赖具体操作系统命令。练习只覆盖可确定、可测试的规则。

## 范围

- `MemorySize`：解析 `B`、`KiB`、`MiB`、`GiB` 并换算字节。
- `ProcessTable`：用进程快照统计运行中进程、状态数量和内存总量。
- `RoundRobinScheduler`：模拟轮转调度的执行顺序。
- `FilePermission`：解析类 Unix 符号权限并转换八进制。
- `EnvironmentVariables`：读取必需变量、默认值和 `PATH` 片段。
