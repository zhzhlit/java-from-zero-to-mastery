# v0.3.11 Java Core Review Design

## Goal

Close the Java Core stage with one learning-ready review lesson and the first runnable flagship project module: a command-line course manager backed by Java 21, Maven, and JUnit 5.

## Scope

- Add `flagship-project` to the root Maven reactor.
- Add `flagship-project/course-manager-cli` as a jar module.
- Model courses, lessons, course levels, course catalog queries, study progress, and local progress persistence.
- Use only Java Core APIs already covered in v0.3: classes, collections, exceptions, file I/O, date-time, Stream, enums, `Optional`, and JUnit.
- Publish `docs/zh-CN/guide/02-java-core/12-java-core-review.md`.
- Update README, roadmap, flagship project docs, previous lesson links, and VitePress navigation.
- Bump Maven project versions to `0.3.11-SNAPSHOT`.

## Out of Scope

- Interactive terminal UI, argument parsing libraries, database persistence, HTTP APIs, Spring Boot, frontend pages, authentication, Docker, Kubernetes, and English documentation parity.
- Rewriting earlier examples or object-oriented exercises.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl flagship-project/course-manager-cli -am test`
- `mvn -B clean verify`
- `npm run check:links --prefix website`
- `npm run docs:build --prefix website`
