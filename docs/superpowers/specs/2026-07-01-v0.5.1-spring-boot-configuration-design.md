# v0.5.1 Spring Boot 配置设计

## 目标

新增 `exercises/spring-boot-configuration` 练习模块和中文课程章节，让学生在 Spring Boot 基础之后掌握 `application.properties`、`@ConfigurationProperties`、类型安全配置、默认值和 Service 使用配置。

## 范围

- `SpringBootConfigurationApplication`：启用配置属性扫描。
- `CourseCatalogProperties`：绑定 `course.catalog` 配置并提供安全默认值。
- `CourseCatalogService`：根据配置生成查询视图并识别特色标签。
- `application.properties`：提供可读的示例配置。
- Binder 与普通单元测试：验证配置绑定和配置驱动的业务行为。

## 验证

- `mvn -B clean verify`
- `npm run check:links --prefix website`
- `npm run docs:build --prefix website`
