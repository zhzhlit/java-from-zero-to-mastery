# v0.3.7 File I/O Design

## Goal

Continue the Java Core stage with one learning-ready file I/O lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `StudyReportFileStore` to `examples/java-core`.
- Demonstrate `Path`, `Files`, UTF-8 text writing, appending, reading, checked `IOException`, and `@TempDir` tests.
- Extend `exercises/object-oriented` with a starter skeleton, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/08-file-io.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.7-SNAPSHOT`.

## Out of Scope

- Binary I/O, serialization, NIO channels, file walking, watch service, stream processing for very large files, and logging frameworks.
- Database persistence, HTTP I/O, and repository abstractions.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
