# v0.4.8 HTTP 基础设计

## 目标

新增 `exercises/http-basics` 练习模块和中文课程章节，让学生在 JDBC 基础之后理解 HTTP 请求、响应、头部、正文、路由分派和响应序列化，为后续 Servlet 与 Web 应用打底。

## 范围

- `HttpRequest` / `HttpResponse`：表示 Web 边界的请求与响应数据。
- `HttpRequestParser`：解析 HTTP/1.1 请求行、请求头和正文。
- `HttpResponseWriter`：写出状态行、响应头、空行和正文。
- `RouteKey` / `SimpleRouter`：按方法与路径分派请求，缺失路由返回 404。
