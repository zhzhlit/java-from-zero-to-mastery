# v0.3.0 Classes and Objects Design

## Goal

Start the Java Core stage with one learning-ready object-oriented lesson backed by Java 21 examples and a starter/solution exercise.

## Scope

- Add `examples/java-core` for class, object, constructor, field encapsulation, instance method, and object collaboration examples.
- Add `exercises/object-oriented` with `starter` and `solution` modules.
- Publish `docs/zh-CN/guide/02-java-core/01-classes-and-objects.md`.
- Connect README, roadmap, previous lesson navigation, and VitePress navigation.

## Out of Scope

- Inheritance, polymorphism, interfaces, records, exceptions as a standalone topic, and collections.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation build before release.
