# v0.3.4 Exceptions Design

## Goal

Continue the Java Core stage with one learning-ready exception handling lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `EnrollmentException` and `EnrollmentService` to `examples/java-core`.
- Demonstrate constructor validation, `throw`, custom unchecked exceptions, targeted `try/catch`, and state safety after failures.
- Extend `exercises/object-oriented` with starter skeletons, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/05-exceptions.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.4-SNAPSHOT`.

## Out of Scope

- Try-with-resources, stack traces in depth, logging frameworks, checked exception design, and global error handling.
- Collections, generics, I/O, records, sealed classes, and pattern matching.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
