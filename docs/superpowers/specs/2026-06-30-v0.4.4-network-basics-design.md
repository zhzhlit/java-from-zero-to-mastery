# v0.4.4 网络基础设计

## 目标

新增 `exercises/network-basics` 练习模块和中文课程章节，让学生在算法入门之后建立 URL、查询参数、HTTP 请求行、状态码和重试边界的第一层工程直觉。

本版本不做真实网络访问、Socket 编程或异步 HTTP 客户端，避免练习受外网和环境波动影响。

## 范围

- `UrlInspector`：解析协议、主机、端口、路径和查询串。
- `QueryParameters`：解析查询参数，覆盖重复键、空值和 URL 解码。
- `HttpStatusClassifier`：识别 1xx 到 5xx 状态码族。
- `HttpRequestLine`：从方法和 URL 构造 HTTP/1.1 请求行。
- `RetryPolicy`：判断幂等方法和临时失败是否适合重试，并计算简单退避。
