# v0.4.9 Servlet 基础设计

## 目标

新增 `exercises/servlet-basics` 练习模块和中文课程章节，让学生在 HTTP 基础之后理解 Servlet 的 service 分派、请求/响应对象、过滤器链和 Dispatcher 映射，为后续真实 Web 容器和 Spring MVC 打底。

## 范围

- `WebRequest` / `WebResponse`：表示 Servlet 边界中的请求上下文和可写响应。
- `MiniServlet`：模拟 `service` 根据方法分派到 `doGet` / `doPost`。
- `ServletFilter` / `FilterChain`：模拟过滤器前后置处理。
- `ServletDispatcher`：按路径找到 servlet，缺失路径返回 404。
