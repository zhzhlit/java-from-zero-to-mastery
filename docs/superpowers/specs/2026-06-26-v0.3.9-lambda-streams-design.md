# v0.3.9 Lambda Streams Design

## Goal

Continue the Java Core stage with one learning-ready Lambda and Stream API lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `StudyStreamAnalyzer` to `examples/java-core`.
- Demonstrate lambda expressions, `Predicate`, `stream`, `filter`, `map`, `sorted`, `toList`, `mapToInt`, `sum`, `max`, `Comparator`, and method references.
- Extend `exercises/object-oriented` with a starter skeleton, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/10-lambda-streams.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.9-SNAPSHOT`.

## Out of Scope

- Parallel streams, custom collectors, advanced `Optional`, spliterators, reactive streams, and performance tuning.
- Enum design, records, annotations, reflection, concurrency, and database queries.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
