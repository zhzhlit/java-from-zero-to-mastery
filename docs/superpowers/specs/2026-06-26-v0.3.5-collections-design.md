# v0.3.5 Collections Design

## Goal

Continue the Java Core stage with one learning-ready collections lesson backed by Java 21 examples and starter/solution practice.

## Scope

- Add `CourseCatalog` to `examples/java-core`.
- Demonstrate `List`, `Set`, `Map`, iteration, duplicate tag handling, key-based lookup, and read-only collection returns.
- Extend `exercises/object-oriented` with a starter skeleton, reference solution, and JUnit tests for the same model.
- Publish `docs/zh-CN/guide/02-java-core/06-collections.md`.
- Connect README, roadmap, previous lesson, and VitePress navigation.
- Bump Maven project versions to `0.3.5-SNAPSHOT`.

## Out of Scope

- Deep generic type design, wildcards, streams, concurrent collections, sorting comparators, and collection performance tuning.
- I/O, records, sealed classes, pattern matching, and database-backed repositories.
- English documentation parity.
- A release tag or GitHub release.

## Validation

- `mvn -B -pl examples/java-core -am test`
- `mvn -B -pl exercises/object-oriented/starter -am compile`
- `mvn -B -pl exercises/object-oriented/solution -am test`
- Root Maven verify and documentation checks before release.
