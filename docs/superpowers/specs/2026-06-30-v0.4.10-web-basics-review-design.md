# v0.4.10 Web 基础综合复盘设计

## 目标

新增 `exercises/web-basics-review` 练习模块和中文课程章节，让学生在 Servlet 基础之后把数据库、JDBC、HTTP 和 Servlet 的边界串成一个小型课程 Web API。

## 范围

- `InMemoryCourseStore`：模拟数据访问边界。
- `ApiRequest` / `ApiResponse`：表示 Web API 输入输出。
- `CourseController`：实现列表、详情和创建接口。
- `AuthFilter`：保护写操作。
- `CourseWebApplication`：按方法和路径分派请求，处理 401、404、405。

## 验证

- `mvn -B clean verify`
- `npm run check:links --prefix website`
- `npm run docs:build --prefix website`
