# v0.5.0 Spring Boot 基础设计

## 目标

新增 `exercises/spring-boot-basics` 练习模块和中文课程章节，让学生在 Web 基础综合复盘之后进入真实 Spring Boot 3 Web 应用，理解启动类、Controller、Service、DTO、JSON 请求响应和 MVC 测试。

## 范围

- `SpringBootBasicsApplication`：最小 Spring Boot 启动类。
- `CourseDto` / `CreateCourseRequest`：请求与响应 DTO。
- `CourseService`：保存课程数据并表达业务边界。
- `CourseController`：提供 `GET` 列表、`GET` 详情和 `POST` 创建接口。
- `MockMvc` 测试：验证状态码、JSON 字段和 `Location` 响应头。

## 验证

- `mvn -B clean verify`
- `npm run check:links --prefix website`
- `npm run docs:build --prefix website`
