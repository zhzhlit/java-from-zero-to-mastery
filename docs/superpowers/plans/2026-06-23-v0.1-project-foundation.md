# Java Learning Platform v0.1 Project Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a publishable v0.1 repository foundation with a Java 21 Maven build, VitePress documentation site, one verified lesson, one runnable example, one tested exercise, contribution files, and GitHub CI/Pages workflows.

**Architecture:** Keep documentation and source code in one repository. A root Maven aggregator validates Java examples and exercises; a self-contained VitePress package under `website/` renders Markdown from `docs/`; GitHub Actions independently verifies Java, documentation, and links before publishing the site.

**Tech Stack:** Java 21, Maven 3.9+, JUnit 5, Node.js 22+, VitePress 1.6.4, GitHub Actions, Markdown

public final class NumberClassifier {

    private NumberClassifier() {
    }

    public static String classify(int number) {
        if (number > 0) {
            return "positive";
        }
        if (number < 0) {
            return "negative";
        }
        return "zero";
    }
}
